01 자기소개

안녕하세요.
디자인 전공의 에이전시 출신 5년차 웹 퍼블리셔 진수정입니다.

저는 전반적인 웹 제작이 가능하며, 반응형 웹사이트와 스크립트를 활용한 동적인 웹사이트 제작에 능숙합니다. 예시 사이트를 그대로 구현할 수 있는 스킬을 보유하고 있으며, 고급 인터랙션과 스크롤 제어 인터랙션을 연구 중입니다.

주로 JavaScript와 jQuery를 다루며, CSS 애니메이션을 깊이 공부하고 있습니다. 또한, 최신 트렌드를 반영하여 3D, Three.js, React 등을 학습하고 있습니다. 디자이너들과 Figma로 협업하여 프로그램에 대한 이해도가 높고, Git을 활용한 개발자와의 협업 경험도 다수 있습니다.

연차가 쌓일수록 고여있지 않기 위해 노력하며, 다양한 클론 코딩을 통해 최신 기술을 습득하고 있습니다. 인공지능을 활용한 코드 작업에도 도전 중입니다. 프로젝트는 혼자 하는 일이 아니기 때문에 소통을 중요하게 생각하며, 효과적인 의견 전달 방법을 고민합니다. 항상 배움의 자세로 임하며, 배운 내용을 응용하여 결과물을 만들어내는 사람이 되겠습니다.

  • HTML을 이용한 마크업
  • SCSS 전처리기 사용
  • Javascript로 인터랙션 구현
  • Webpack을 통한 최적화 빌드
  • PHP를 이용한 CRUD 구현
  • SourceTree를 이용한 Github 사용
  • 피그마
  • 포토샵
  • 일러스트
