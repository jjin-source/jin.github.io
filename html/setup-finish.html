01 자기소개

저는 디자이너 출신의 퍼블리셔 입니다. 그래서 저는 심미적이면서 동시에 접근하기 쉬운 웹사이트를 만드는 것이 목표입니다. 처음 사용자가 웹사이트를 사용할때 저는 사용자가 다양한 경험을 할 수 있는 웹사이트를 만드는것을 목표로 하고있습니다. 그래서 저는 UX 개발자가 되기 위한 방향으로 연구 및 자기계발을 하고 있습니다.

  • HTML을 이용한 마크업
  • SCSS 전처리기 사용
  • Javascript로 인터랙션 구현
  • Webpack을 통한 최적화 빌드
  • PHP를 이용한 CRUD 구현
  • SourceTree를 이용한 Github 사용
  • 피그마
  • 포토샵
  • 일러스트
